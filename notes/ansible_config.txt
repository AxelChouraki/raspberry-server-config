	Installer ansible sur un linux (wsl par exemple)

sudo apt update
sudo apt install software-properties-common
sudo apt-add-repository --yes --update ppa:ansible/ansible
sudo apt install ansible

	Récupérer la configuration
git clone ?
or
sudo rsync -r /mnt/c/Users/sramy/OneDrive/Documents/GitHub/raspberry-server-config/* /etc/ansible

	Configurer les cibles si ce n'est pas déjà fait
ajouter à /etc/ansible/hosts la ligne :
192.168.1.42 ansible_user=sramy
vérifier que l'host a bien été pris en compte :
ansible all --list-hosts

	Test de connexion
Si besoin de password:
	ansible all -m ping -u sramy -k --ssh-common-args='-o StrictHostKeyChecking=no'
sinon:
	ansible all -m ping
Pour run un seul playbook : 
ansible-playbook /etc/ansible/playbooks/configure_miscellaneous_install.yml -k

	Lancer 

ATTENTION : ANSIBLE CONFIGURER POUR UTILISER SYSTEMCTL ET NON SYSTEMD
Télécharger le dossier ansible
commentez les lignes non voulues dans run_playbooks.sh
Vérifier le port (22 pour un premier run, 2002 sinon) et les hosts dans le fichier hosts
sudo ./run_playbooks.sh

	SI PROBLEME DE CHECK SSH KEY
-o StrictHostKeyChecking=no

	SI PROBLEME DE SSH CONFIGURÉ
changer le port dans hosts
